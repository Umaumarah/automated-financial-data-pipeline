# Automated Financial Data Pipeline

This project automatically fetches financial data (stocks, crypto, forex), cleans it, stores it in SQLite, and generates plots using Python.

## Features
- Fetch data from Yahoo Finance
- Clean and transform data
- Store data in SQLite database
- Generate and save plots (closing price trends, comparisons)

## Folder Structure
financial-data-pipeline/
├── data/ # Raw and cleaned CSVs, SQLite DB
├── plots/ # Saved plot images
├── scripts/ # Python scripts: fetch, transform, load, visualize
├── run_pipeline.py # Main script to automate the pipeline
├── automate_pipeline.py
└── README.md
