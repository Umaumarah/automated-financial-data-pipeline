# Automated Financial Data Pipeline

This Python project automates the fetching, cleaning, storage, and visualization of financial data such as stocks, crypto, and forex.

## Features
- Fetch data from Yahoo Finance using `yfinance`
- Clean and transform raw data
- Store data in SQLite database
- Generate plots for trends and comparisons
- Fully automated pipeline via `run_pipeline.py`

## Folder Structure

financial-data-pipeline/
├── data/ # Raw & cleaned CSVs, SQLite DB
├── plots/ # Saved plot images
├── scripts/ # Python scripts: fetch, transform, load, visualize
├── run_pipeline.py # Main script to run the full pipeline
├── automate_pipeline.py
├── requirements.txt
└── README.md

## Installation
1. Clone the repository:
```bash
git clone https://github.com/<your-username>/financial-data-pipeline.git
cd financial-data-pipeline

Install dependencies:
pip install -r requirements.txt

How to Run
Simply run the pipeline:
python run_pipeline.py

This will:
Fetch latest data for predefined symbols
Transform and clean the data
Store it in SQLite database
Generate plots in the plots/ folder

Notes:
Raw and cleaned CSV files are stored in the data/ folder.
Plots are automatically saved in the plots/ folder.
Modify the symbols list in run_pipeline.py to track different financial instruments.
