# Automated Financial Data Pipeline

This Python project automates the fetching, cleaning, storage, and visualization of financial data such as stocks, crypto, and forex.

## Features
- Fetch data from Yahoo Finance using `yfinance`
- Clean and transform raw data
- Store data in SQLite database
- Generate plots for trends and comparisons
- Fully automated pipeline via `run_pipeline.py`

## Folder Structure

financial-data-pipeline/
├── data/ # Raw & cleaned CSVs, SQLite DB
├── plots/ # Saved plot images
├── scripts/ # Python scripts: fetch, transform, load, visualize
├── run_pipeline.py # Main script to run the full pipeline
├── automate_pipeline.py
├── requirements.txt
└── README.md
